# Glossary
| Term | Definition |
| ---- | ---------- |
| `Actor` | An entity with some agency. |
| `Person` | An actor consisting in a single individual. |
| `Organization` | An actor consisting in a group of individuals. |
| `Event` | A process that is defined and delimited in time. |
| `Appellation` | A human-readable label attached to something in order to name it. |
| `Identifier` | A label attached to something in order to uniquely identify it.
| `Type` | A category something is an instance of. |
| `Attribute Assignment` | An event that represents the assignment of a characteristic to something. |
| `Place` | A geographical location. |
| `is identified by` | This property links a place with an appellation. |
| `has type` | This property links something to a type. |
| `has content` | This property attaches textual content to an appellation. |
| `has start time` |  |
| `has end time` |  |
| `full name` |
| `first name` |
| `last name` |
| `birth` | A type of event that represents the birth of a person. |
| `death` | A type of event that represents the death of a person. |
